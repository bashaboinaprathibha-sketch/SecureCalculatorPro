Secure Calculator Pro

 Project Overview

Secure Calculator Pro is a Python-based menu-driven calculator application that performs basic arithmetic operations while implementing robust exception handling, input validation, logging, file handling, and object-oriented programming concepts.

This project demonstrates the use of Python programming fundamentals and best coding practices.

Features

- Addition
- Subtraction
- Multiplication
- Division
- Input Validation
- Exception Handling
- Custom Exceptions
- Error Logging with Timestamps
- Calculation History
- Analytics Report
- Menu-Driven Interface
- Object-Oriented Programming (OOP)

 Technologies Used

- Python 3
- Object-Oriented Programming (OOP)
- Exception Handling
- File Handling
- Logging Module

 How to Run

1. Download or clone the repository.
2. Open the project folder in Visual Studio Code.
3. Open the terminal.
4. Run the following command:

```bash
python main.py
```

or

```bash
py main.py
```

Menu Options

1. Addition
2. Subtraction
3. Multiplication
4. Division
5. View Calculation History
6. View Error Report
7. View Analytics
8. Exit

 Files Generated

history.txt

Stores all successful calculations.



10 + 20 = 30
50 - 15 = 35
12 * 5 = 60
100 / 4 = 25

error.log

Stores all errors with timestamps.

 Analytics Report

Displays:

- Total Calculations
- Total Errors
- Most Common Error Type
